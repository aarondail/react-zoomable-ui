After a press starts there are some cases where it can be canceled rather than ended. For example when on finger starts touching the screen that will trigger onPressStart. If a second finger starts touching the screen though, that will cause the press to be "cancelled". A press is effectively a single finger touch or mouse/track-pad click and drag motion.
